# Passgen.sh - simple readable password generator


Install: `sudo bash -c "source install.sh"`

Usage: $ `source passgen.sh` 
Or add it to .bashrc:
```
echo "source `pwd`/passgen.sh" >> ~/.bashrc
```

Then just use `passgen`.

Example:


```
$ passgen
rhic4HujNirdobhijmHaHejNiUgramtaHGrawmpu
```
